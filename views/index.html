Description
Nav2D is a set of libraries and ROS nodes allows to control a mobile robot within a planar environment. The main features are a purely reactive obstacle avoidance, a simple path planner, and a graph based SLAM (Simultaneous Localization and Mapping) node that allows multiple robots to cooperatively build a map. By using an exploration plugin for the navigation module, one or more robots can autonomously explore there working space and create a shared map for navigation on all robots.
